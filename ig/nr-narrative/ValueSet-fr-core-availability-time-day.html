ValueSet: First day of the workweek | Premier jour de la semaine de travail

Official URL: http://hl7.fr/ig/fhir/core/ValueSet/fr-core-availability-time-day Version: 2.0.0
Standards status: Informative Maturity Level: 1 Computable Name: FRCoreValueSetAvailabilityTimeDay

Specifies the day on which the workweek starts. Valid values are MO, TU, WE, TH, FR, SA and SU. Spécifie les jours à partir desquels la semaine de travail commence

References

This value set is not used here; it may be used elsewhere (e.g. specifications and/or implementations that use this content)
