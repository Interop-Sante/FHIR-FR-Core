Data Type Profile: FR Core Human Name Profile

Official URL: http://hl7.fr/ig/fhir/core/StructureDefinition/fr-core-human-name Version: 2.0.0
Draft as of 2023-09-19 Computable Name: FRCoreHumanNameProfile

French profile of datatype HumanName with constraints on prefix and suffix Profilage du type de données HumanName pour prise en compte de la civilté au niveau de l’élément prefix et du titre au niveau de l’élément suffix

Usage:

  • Use this DataType Constraint: FR Core Patient Profile, FR Core Practitioner Profile and FR Core Related Person Profile
